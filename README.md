# Expense Tracker 💰

A professional expense tracking web application built with Laravel & AdminLTE.

## Features
- Authentication (Login/Register)
- Manage Expenses & Categories
- Dashboard with Statistics
- REST API with Laravel Sanctum
- Bilingual Support (Arabic/English)

## Tech Stack
- Laravel 11
- AdminLTE 3
- MySQL
- Laravel Sanctum

## Installation
- clone the repo
- composer install
- cp .env.example .env
- php artisan key:generate
- php artisan migrate --seed
- php artisan serve
